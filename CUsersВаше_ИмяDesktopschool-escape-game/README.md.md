# 🏫 СХРОНКА: Школьный кошмар

## Игра-побег от учителя!

### 🎮 Управление
- **WASD** или **стрелки** - движение
- **Кнопки на экране** - для телефона

### 🎯 Цель
Собрать 3 шпаргалки и добежать до выхода, не попавшись учителю!

### 🚀 Запуск
1. Откройте `index.html` в браузере
2. Или запустите сервер: `python -m http.server 8000`
